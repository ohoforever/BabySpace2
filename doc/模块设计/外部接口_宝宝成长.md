##查询宝宝成长列表

###查询宝宝成长列表接口

家长通过微信查询自己宝宝的成长列表


http请求方式: post

前置条件：用户必须先登录，

    http://ip:port/if/babymature/queryBabyMatureList


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },        
        "body":
        {
            "unionId":"wzreildknf90454kj434",
            "childId":"wzr434",
	    "maxDate":"2015-09-20"
        }

    } 


参数|是否必须|说明
----|----|-----
appId|是|应用ID
timestamp|是|时间戳
nonce|是|随机数
signature|是|签名值,MD5(按值的字典顺序排列组合成字符串(appId,appKey,app.nonce,app.timeStamp))
unionId|是|微信号
childId|否|宝宝id
maxDate|否|显示最大日期,为空时取最新的,有值时,取小于该值的数据,并按由大到小取几条

###查看宝宝详情接口

家长通过微信查看宝宝的某一成长详情


http请求方式: post


    http://ip:port/if/babymature/getBabyMature


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },        
        "body":
        {
            
            "matrueId":"rwsdsi123"
        }
    } 


参数|是否必须|说明
----|----|-----
appId|是|应用ID
timestamp|是|时间戳
nonce|是|随机数
signature|是|签名值,MD5(按值的字典顺序排列组合成字符串(appId,appKey,app.nonce,app.timeStamp))
matrueId|是|成长id




2）返回说明

正常时的返回JSON数据包示例：
 
    {"errcode":0,"errmsg":"查看宝宝详情成功",
	"result":{
			"matrueId":"rwsdsi125",
			"images:["http://7niu.com/ewrw93d","http://7niu.com/rt5rr54t"]
			
		}
    }

错误时的JSON数据包示例：

    {"errcode":10000,"errmsg":"查看宝宝详情失败，其它错误"}


###添加评论接口

家长通过该接口为某一成长事件添加评论


http请求方式: post

    http://ip:port/if/babymature/addComment


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },        
        "body":
        {
            
            "unionId":"wzreildknf90454kj434",
	    "matrueId":"rwsdsi125",
	    "replyTo":"钉钉老师",
	    "comment":"谢谢老师"
        }
    } 


参数|是否必须|说明
----|----|-----
appId|是|应用ID
timestamp|是|时间戳
nonce|是|随机数
signature|是|签名值,MD5(按值的字典顺序排列组合成字符串(appId,appKey,app.nonce,app.timeStamp))
unionId|是|家长微信号 
matrueId|是|成长记录id 
replyTo|否|回复对象,如为空,则是发送评论
comment|是|评论内容




2）返回说明

正常时的返回JSON数据包示例：
 
    {"errcode":0,"errmsg":"添加评论成功"}


错误时的JSON数据包示例：

    {"errcode":10000,"errmsg":"添加评论失败，其它错误"}
