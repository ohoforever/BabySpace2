#接口

##查询广告列表

###查询广告列表接口

家长通过微信查询自己宝宝的成长列表


http请求方式: post

前置条件：用户必须先登录，

    http://ip:port/if/advertisement/getAdvertisementList


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },
        "body":
        {
            "unionId":"wzreildknf90454kj434",
            "pageIndex":0,
	    "pageSize":10
        }

    } 


参数|是否必须|说明
----|----|-----
appId|是|应用ID
timestamp|是|时间戳
nonce|是|随机数
signature|是|签名值,MD5(按值的字典顺序排列组合成字符串(appId,appKey,app.nonce,app.timeStamp))
unionId|是|微信号
pageIndex|是|显示页数
pageSize|是|每页显示条数


2）返回说明

正常时的返回JSON数据包示例：
 
    {"errcode":0,"errmsg":"查询广告列表成功","pageSize":10,"pageIndex":0,"total":100,
	"list":[
			{
				"insertTime":"2015-09-19 08:21:28",
				"description":"水果大平卖",
				"updateTime":"2015-09-19 08:21:31",
				"id":"1",
				"tag":"水果",
				"title":"水果大平卖",
				"type":"ZAJ",
				"content":"",
				"url":null,
				"status":"OK#"
			}
		]
	}
    }

错误时的JSON数据包示例：

    {"errcode":10000,"errmsg":"查询广告列表失败，其它错误"}

参数|是否必须|说明
----|----|-----
list|否|结果列表
title|是|标题
description|是|摘要
id|是|广告id
type|是|类型,ZAJ早教
content|是|广告内容
url|是|连接地址
status|是|状态,OK#正常


###查看广告详情接口

家长通过微信查看某个广告详情


http请求方式: post


    http://ip:port/if/advertisement/getAdvertisementDetail


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },        
        "body":
        {
            
            "advertisementId":"rwsdsi123"
        }
    } 


参数|是否必须|说明
----|----|-----
appId|是|应用ID
timestamp|是|时间戳
nonce|是|随机数
signature|是|签名值,MD5(按值的字典顺序排列组合成字符串(appId,appKey,app.nonce,app.timeStamp))
advertisementId|是|记录id




2）返回说明

正常时的返回JSON数据包示例：
 
    {"errcode":0,"errmsg":"查看广告详情成功",
	"result":{
			"insertTime":"2015-09-19 08:21:28",
			"description":"水果大平卖",
			"updateTime":"2015-09-19 08:21:31",
			"id":"1",
			"tag":"水果",
			"title":"水果大平卖",
			"type":"ZAJ",
			"content":"快来看,快来看,好便宜的水果,苹果2块一节",
			"operator":"1",
			"url":null,
			"status":"OK#"
			
		}
    }

错误时的JSON数据包示例：

    {"errcode":10000,"errmsg":"查看广告详情失败，其它错误"}
